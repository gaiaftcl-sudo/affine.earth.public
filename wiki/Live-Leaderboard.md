## 4. How to refresh this page’s measured section

```bash
cd llm-llvm-benchmark-suite
python3 -m pytest tests/ -v
python3 scripts/verify_real_numbers_no_flub.py
python3 -m llm_llvm_bench.cli.main llvm run --opt-levels -O0,-O2,-O3,-Os --compiler clang \
  --out reports/llvm_benchmark_live.json
python3 scripts/run_live_affine_earth_benchmark.py
```

Then update the MEASURED tables from the new JSON/Markdown.
