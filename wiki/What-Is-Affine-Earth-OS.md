# What is Affine.Earth OS

Affine.Earth OS is a **sovereign wallet operating surface** with a live **language-game** membrane. Identity is an edge wallet — not email/password. After Create wallet + QFOT, the app opens onto Franklin chat, Docs, Profile, and a catalog of **12 LIVE Games**.

This page is product identity. For the signup walkthrough video, go once to [Create account](Create-Account-Signup). For proof that Games answer live, go to [In action](In-Action) or the [Home](Home) hero.

## What the OS is for

1. **Sovereignty** — edge wallet session; export your key; no custodial signup form.
2. **Language games** — Wittgenstein-style turns in Games (linguistic membrane, formal manifold, coding, torsion, domain cards).
3. **Economics surface** — Profile genesis QFOT when onboard succeeds (**PROVEN 100/1** in the 2026-07-20 UI re-verify).
4. **Evaluation target** — the same live surface agents probe via healthz + (when ready) OpenAI-compatible JSON endpoints for harnesses.
